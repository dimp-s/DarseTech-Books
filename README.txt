This project is done using Node.js and express.js api
Lowdb database was used as local database to minimize development time (consider using an earlier version!)

The project uses Swagger APi documentation using swaggerUI and swagger-jsdoc npm packages.
Steps to run:
1. goto project directory and install necessary packages using terminal through command
    npm install
2. After insatllation of packages start the server using 
    node index.js
3. Open browser and goto 
    http://localhost/5000/api-docs 
4. Try and test the APIs as needed
